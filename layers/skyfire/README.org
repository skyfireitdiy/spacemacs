* Description
 skyfire layer 包含了以下组件
 - company-tabnine
  基于深度学习的自动完成插件
 - graphviz-dot-mode
  graphviz dot 语言支持
 - counsel-etags
  etags 跳转支持
 - format-all
  格式化支持
 - ob-go 
  org-mode 中对 go 代码的执行支持
 - paredit
  对括号编辑的优化
* Install
To use this configuration layer, add it to your =~/.spacemacs=. You will need to
add =skyfire= to the existing =dotspacemacs-configuration-layers= list in this
file.
